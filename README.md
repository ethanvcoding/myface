# My Face Everywhere

This Chrome extension replaces all images and videos with images of yourself. All the action happens locally, and your data is never sent to Google or any websites. Just follow the instructions below to get it all set up! It even disguises itself as a print utility, if you want to prank your buddy.

# Setup

## 1. Download

a) Click the green "Code" button  
![Step 1a](.blob/img/md/readme/screenshot/4821937/1/a/jpg/image.jpg)

b) Click "Download ZIP"  
![Step 1b](.blob/img/md/readme/screenshot/9157264/1/b/jpg/image.jpg)

c) Go into your file explorer and unzip the file (double-click)  
![Step 1c](.blob/img/md/readme/screenshot/3071842/1/c/jpg/image.jpg)

## 2. Add Your Images

a) Put images into the image folder (max: 6)  
![Step 2a](.blob/img/md/readme/screenshot/7642095/2/a/jpg/image.jpg)

b) Rename to image#.png  
![Step 2b](.blob/img/md/readme/screenshot/1289376/2/b/jpg/image.jpg)

c) Open Chrome  
![Step 2c](.blob/img/md/readme/screenshot/5938417/2/c/jpg/image.jpg)

## 3. Import Into Chrome

a) Navigate to `chrome://extensions`  
![Step 3a](.blob/img/md/readme/screenshot/8471629/3/a/jpg/image.jpg)

b) Turn on Developer Mode  
![Step 3b](.blob/img/md/readme/screenshot/2645931/3/b/jpg/image.jpg)

c) Click "Import Unpacked"  
![Step 3c](.blob/img/md/readme/screenshot/7392058/3/c/jpg/image.jpg)

d) Select the extension directory  
![Step 3d](.blob/img/md/readme/screenshot/1564823/3/d/jpg/image.jpg)

## 4. Enjoy!

a) Start browsing and see your face everywhere!  
![Step 4a](.blob/img/md/readme/screenshot/9083174/4/a/jpg/image.jpg)
